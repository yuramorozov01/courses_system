## About The Project

[![Courses screenshot][product-screenshot]](https://github.com/yuramorozov01/courses_system)

This web-application provides API for a courses system. 
* Every user can create course (user automatically becomes a teacher of this course).


* Every teacher has the next features: 
  * Add/delete other users as a students/teachers.
  * Create/update/delete lectures with files.
  * Create/update/delete task statement with files for the lecture.
  * See added tasks by students.
  * Create/update/delete marks to added tasks.
  * Create/update/delete messages attached to the mark.
  * See messages of the task's author attached to the mark.


* Every student has the next features:
  * See courses.
  * See lectures of courses, where user is a student of course.
  * See task statements with attached files of specified lecture.
  * Create/update/delete completed task with files to check by a teacher.
  * See marks of task that had been sent.
  * Create/update/delete messages of student's tasks.
  * See messages in student's tasks that had been sent by a teacher.

For more details you can see API documentation, generated by Swagger (how to launch - see in the sections below).

## Getting Started

### Prerequisites

To run this program you have to follow this steps:

* Install [Docker Compose](https://docs.docker.com/compose/)

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/yuramorozov01/courses_system.git
   cd courses_system/
   ```
2. Create .env file in the root folder (near docker-compose.yaml).
   ```
   SECRET_KEY=Secret key of your application
   DEBUG=True if want to turn on Debug mode (else - False)
   
   DB_ENGINE=Engine for database
   DB_NAME=Database name
   DB_USER=Database user
   DB_PASSWORD=Database user password
   DB_HOST=Database host
   DB_PORT=Database port
   ```
3. Give execute root to run bash scripts
   ```sh
   chmod +x run.bash
   chmod +x createsuperuser.bash
   ```

<!-- Executing EXAMPLES -->
## Executing
1. Run Docker (launch Docker Desktop)
2. Run script
   ```sh
   ./run.bash
   ```
3. Create superuser: 
   ```sh
   ./createsuperuser.bash
   ```
4. Open the following URL to see API documentation
   ```
   127.0.0.1:8000/swagger/
   ```
